1.Project Name:
DigiTools – Empowering Your Digital Workflow

2.Description:
DigiTools is a premium digital marketplace that lets users browse, purchase, and manage a curated collection of digital tools — including AI-powered software, design assets, templates, and productivity apps — all in one place. It features a shopping cart system, pricing plans, and a clean modern interface designed to help creators and professionals work smarter. The project emphasizes a "mobile-first" approach, ensuring a seamless experience across all modern web browsers and devices.

3.Technologies Used:
Frontend  : React.js (v19.0.0)
Styling   : Tailwind CSS + DaisyUI
Icons     : React Icons
Build Tool: Vite
JavaScript (ES6+), React Toastify
Deployment: Netlify

4.3 Key Features of this Project: 
Feature 1 — Product Catalog with Dynamic Cart System:
Users can browse a list of premium digital products. Each product can be added to the cart, which updates the cart counter in the navbar in real time and shows toast notifications on every action (add, remove, checkout).

Feature 2 — Pricing Section with Interactive Cards:
A dedicated pricing section displays three subscription tiers. Each cart with a hover animation that transitions the card to a gradient background — giving users a visually engaging way to compare and select plans.

Feature 3 — Adaptive Layout System: 
The project features a fully fluid grid system that automatically adjusts its layout for smartphones, tablets, and desktops, providing a professional and consistent look regardless of the user's device.
